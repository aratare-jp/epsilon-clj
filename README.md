# `epsilon-clj`

[![codecov](https://codecov.io/gh/aratare-jp/epsilon-clj/branch/master/graph/badge.svg?token=RJCEPYBF3I)](https://codecov.io/gh/aratare-jp/epsilon-clj)

various goodness.

# Status
`ALPHA`

# Features

- [x] XML model
- [ ] EDN model
- [ ] JSON model
- [x] Hot-reload on file changes
- [ ] REPL
